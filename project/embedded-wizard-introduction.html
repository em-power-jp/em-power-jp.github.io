Embedded Wizard
Write Once, Run Anywhere

無駄のない、高拡張性、素早い GUIアプリ開発

Embedded Wizardとは

Embedded Wizardとは、組込みソフト専門集団 Tara Systems社が開発した組込みシステム専用 GUI開発ツールです。

特徴

総合開発環境
  • WYSIWYGによる、直観的なアプリ開発
  • PNG、JPG、BMP、GIF画像データの取り込み
  • オブジェクトとコンポーネント指向開発
  • GUIアプリ開発に最適なされた Chora言語サポート
  • プロトタイピングやデバッガ
  • メモリ使用量、自動生成コードのサイズを可視化
  • ドキュメントジェネレーター
リッチ GUI
  • 数多くのUIコンポーネント提供
  • ベクターグラフィックスのサポート
  • 多国語、製品バリエーションに対応
  • UNICODE対応
  • TrueTypeフォントエンジン
  • ガベージコレクション
  • 数多くのサンプルアプリケーションを提供
カラーフォーマット
  • カラーフォーマット(RGBA8888, RGB888, RGBA4444, RGB565)
  • 8ビット CLUT(Color Look-up Table) カラーフォーマット(Index8)
  • 8ビット グレースケール　モノクロームフォーマット(LumA44)
その他
  • ARM NEON、OpenGL ES2.0サポート

対応 CPU/OS

  • STマイクロエレクトロニクス制

           STM32F4、STM32F7、STM32H7、STM32L4、STM32MP1 シリーズ

  • NXPセミコンダクターズ制

           LPCXpressoシリーズ、i.MX RTシリーズ

  • ルネサスエレクトロニクス制

           RX65N、RZ/A1H

  • マイクロチップテクノロジー制

           PIC32シリーズ、SAMA5D2シリーズ

  • Espressifシステムズ制

           ESP32ファミリー

  • ブロードコム制

           Raspberry Pi 3、4

  • Google Android、Javascript/WebGL
